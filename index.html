Mech Lab — Automated Engineering Education Pipeline

Solo developer · 2026
Built a fully automated video production pipeline that turns a topic outline into a publish-ready YouTube lecture. Steps include scripting, narration, animation, and upload without manual handoff between stages. The system generates Claude API scripts, synthesizes narration through ElevenLabs, renders synchronized Manim animations using a beat-structured timing engine, and assembles everything via FFmpeg. All steps are managed and controlled through a 2000+ line Python script guiding the pipeline process, capable of generating educational videos for any topic.
Python Claude API ElevenLabs Manim FFmpeg
